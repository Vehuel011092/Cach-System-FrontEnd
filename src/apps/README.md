# Apps internas

Esta carpeta se utilizará para organizar los módulos/aplicaciones del sistema CACH
(por ejemplo: control-escolar, servicios, reportes, etc.).

Estructura sugerida:

- `src/apps/<nombre-app>/components`
- `src/apps/<nombre-app>/pages`
- `src/apps/<nombre-app>/services`

> Se irá poblando conforme se construyan nuevos componentes del sistema.
